### Start The Project
____________________

Note: You must have Node installed. 

Install Node here: [https://nodejs.org/en/](https://nodejs.org/en/)

Clone the repo

```shell
git clone https://github.com/eric38383/buyrent.git
```

Go to Directory

```shell
cd buyrent
```

Install Dependecies

```shell
npm install
```

Start App

```shell
npm start
```
